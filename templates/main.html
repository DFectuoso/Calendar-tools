Calendar Tools:

This is a very simple webservice API to work with calendr stuff like iCal.
You can use this API by posting to the next urls

Find ical file in the html of a website.

Url: http://calendar-tools.appspot.com/find-ical
Method: POST
Variables: url, the url of the html you want the api to parse.
Answer: a plaintext of the url of the ical file.
Example:

Parse an ical file in a especific url.

Url: http://calendar-tools.appspot.com/parse-ical
Method: POST
Variables: url, the url of the ical file.
Answer: a xml representation of the ical file
Example:

Find and Parse an ical file in a especific url.

Url: http://calendar-tools.appspot.com/find-and-parse-ical
Method: POST
Variables: url, the url of the html you want the api to parse.
Answer: a xml representation of the ical file found in the file
